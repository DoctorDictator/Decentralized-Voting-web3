# Decentralized Voting

A production-grade decentralized voting DApp built with Solidity, Hardhat, React, and ethers.js v5.

## Features

### Smart Contract
- Role-based access (owner, members)
- Vote creation with 2–8 options, deadlines, and quorum
- Vote delegation (one vote per member per vote)
- Vote cancellation by owner before expiry
- Pausable by owner for emergency stops
- Reentrancy protection on critical functions
- Custom Solidity errors for gas-efficient reverts
- Rich indexed events for off-chain indexing

### Frontend
- Dark glassmorphism design system with CSS custom properties
- Wallet integration via MetaMask (account + network detection)
- Real-time vote list updates via event subscriptions
- Inline metadata builder (`data:` URI) — no IPFS keys needed
- Filter (All / Active / Ended), search, and sort (newest / oldest / most votes)
- Vote detail modal with animated progress bars
- Winner highlighting on expired votes
- Live countdown timers
- Toast notification system (success / error / pending / info)
- Skeleton loaders during data fetch

## Getting Started

### Prerequisites
- [Node.js](https://nodejs.org/) v18+
- [MetaMask](https://metamask.io/) browser extension

### Backend Setup

```bash
cd backend
npm install
```

#### Run Tests

```bash
npx hardhat test
```

#### Deploy to Local Hardhat Node

Terminal 1 (node):
```bash
npx hardhat node
```

Terminal 2 (deploy):
```bash
npx hardhat run --network localhost scripts/deploy.js
```

The deploy script automatically copies the contract ABI and address to `react_frontend/src/abi/Voting.json`.

### Frontend Setup

```bash
cd react_frontend
npm install
npm start
```

Open [http://localhost:3000](http://localhost:3000) and connect MetaMask to `localhost:8545`.

### Deploy to Sepolia (Optional)

Set environment variables:
```bash
export SEPOLIA_RPC_URL=https://sepolia.infura.io/v3/YOUR_KEY
export PRIVATE_KEY=your_wallet_private_key
```

Then deploy:
```bash
npx hardhat run --network sepolia scripts/deploy.js
```

## Usage

1. **Connect Wallet** — Click "Connect Wallet" in the navbar
2. **Become a Member** — Click "Become Member" to join the voting community
3. **Create a Vote** — Navigate to "Create", fill in description, options (2–8), end time, and optional quorum
4. **Vote** — Browse active votes, click an option to vote, or click a card to see the detail modal
5. **Cancel** — Vote owners can cancel their own vote before it expires
6. **Filter & Search** — Use the filter bar to find votes by status or search by description

## API

### Contract Functions

| Function | Description |
|---|---|
| `join()` | Register as a voting member |
| `createVote(uri, endTime, options, quorum)` | Create a new vote |
| `vote(voteId, option)` | Cast a vote on an option |
| `cancelVote(voteId)` | Cancel own vote before expiry |
| `getVote(voteId)` | Get full vote details |
| `getVoteCount()` | Total number of votes created |
| `didVote(member, voteId)` | Check if member voted |
| `pause()` / `unpause()` | Owner-only emergency stop |
